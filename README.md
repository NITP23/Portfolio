# Personal Portfolio

Welcome to my personal portfolio project! This is a modern, responsive single-page web application built to showcase my skills, projects, and professional experience.

## 🚀 Technologies Used
This project is built using a modern frontend stack:
- **React.js** - For building the user interface
- **Vite** - As the build tool and development server for lightning-fast compilation
- **CSS3 / AOS** - For styling and beautiful scroll animations

## 📁 Project Structure
The main React application is located in the `portfolio` directory.
- `portfolio/src/components` - Contains reusable UI components (Header, HeroSection, About, Projects, etc.)
- `portfolio/src/assets` - Images and other static assets
- `portfolio/index.html` - The main entry point

## 🛠️ How to Run Locally

If you'd like to run this project on your own machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/NITP23/Portfolio.git
   cd Portfolio/portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open in Browser:**
   Once the server starts, open your browser and navigate to `http://localhost:5173` (or the URL provided in your terminal).

## 🌟 Features
- Fully responsive design (works great on desktop, tablet, and mobile)
- Smooth scrolling and entrance animations
- Modular component structure for easy maintenance and scaling
